MEU PORTFÓLIO

Análise Exploratória e Visualização das Vendas por Categoria de Produtos

Este projeto analisa dados de vendas por categoria de produtos, utilizando pandas, matplotlib e seaborn para gerar estatísticas descritivas e visualizações. Ele examina a distribuição das vendas com um histograma e destaca as 10 categorias mais vendidas com um gráfico de barras. O relatório final oferece insights sobre o desempenho das categorias e recomendações de marketing baseadas nas tendências observadas.

Previsão dos Preços das Ações do Walmart Usando ARIMA

Este projeto utiliza o modelo ARIMA para analisar e prever os preços de fechamento das ações do Walmart. Após preparar e visualizar os dados históricos, o modelo é ajustado, e os resíduos são analisados para verificar a adequação. As previsões indicam uma tendência de alta nos preços das ações. O relatório final destaca a importância de considerar fatores externos, como relatórios financeiros e indicadores econômicos, para uma análise completa e precisa.

Predição de Infecção por AIDS com Regressão Logística

Este projeto desenvolve um modelo de Regressão Logística para prever a infecção por AIDS utilizando dados de saúde e demográficos. A análise exploratória identifica padrões nas variáveis, e o pré-processamento prepara os dados para modelagem. Após treinar o modelo, as métricas de avaliação mostram um desempenho razoável. O relatório sugere melhorar o modelo com dados adicionais, como informações genéticas e histórico de exposição, para aumentar a precisão e a capacidade de generalização.

Análise das Pontuações de Linguagens de Programação ao Longo do Tempo

Este projeto analisa a popularidade semanal de Python, Java e C++ ao longo do tempo. Utiliza visualizações, como histogramas e gráficos de dispersão, para identificar tendências e distribuições. Estatísticas descritivas e correlações entre as linguagens são calculadas, destacando possíveis influências externas e sugerindo investigações futuras para compreender melhor as variações observadas nas pontuações.

Análise e Classificação de Reclamações de Consumidores por Categoria de Produto

O projeto consiste em analisar e classificar reclamações de consumidores em categorias de produtos usando técnicas de processamento de linguagem natural e aprendizado de máquina, visando identificar preocupações-chave e fornecer insights para aprimoramentos.

Análise Estatística e Modelagem de Encargos de Seguro de Saúde

O projeto realiza uma análise detalhada dos encargos de seguro de saúde, explorando a relação entre idade, IMC, hábito de fumar, sexo e número de filhos dos segurados. Também inclui modelagem estatística para prever os encargos com base nessas variáveis, proporcionando insights para otimizar a estratégia de preços em uma seguradora de saúde.

Projeto de Análise de Dados sobre Ética em Inteligência Artificial

Neste projeto, realizamos uma análise abrangente de dados sobre ética em Inteligência Artificial (IA), utilizando um conjunto de dados de artigos disponíveis. Exploramos tendências ao longo do tempo, principais tópicos em abstracts, sentimentos expressos e outros insights relevantes para entender as preocupações e tendências relacionadas à ética em IA com base nos artigos analisados.

Análise de Comandos do Kubectl em Kubernetes

Neste projeto, realizamos uma análise abrangente dos comandos do kubectl em Kubernetes, destacando contagens por objetivo, sintaxe e uso de bandeiras. Identificamos padrões de uso, áreas de foco e insights relevantes para especialistas em Kubernetes, fornecendo uma visão abrangente da utilização e distribuição desses comandos essenciais no ambiente de contêineres.

Análise Exploratória de Dados com Visualização Gráfica Dinâmica

Este projeto oferece uma análise exploratória de dados com visualizações gráficas dinâmicas. Ele inclui detecção e tratamento de dados faltantes, estatísticas descritivas e uma variedade de gráficos como histogramas, barras, pizza, linha, dispersão, boxplot e matriz de correlação. Também permite uma análise especializada para insights adicionais.
